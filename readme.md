# Random Face Points Generator (Jittered Grid Edition)

選択した面（Face）の上に、ランダムなガイドポイント（cpoints）を生成する SketchUp プラグインです。
通常の純粋なランダム配置とは異なり、格子（グリッド）をベースにした「揺らぎ」を加えることで、密度を保ちつつ自然なばらつきを実現しています。

## 💡 開発の背景
純粋なランダム（ポアソン分布）で点を生成すると、場所によって極端な密集や空白が生まれてしまいます。本プラグインでは「格子揺らぎ法（Jittered Grid）」を採用することで、ボロノイ図の母点や植栽の配置ガイドとして使いやすい、均一かつランダムな分布を可能にしました。

## 🚀 主な特徴
* **格子揺らぎ法 (Jittered Grid)**: 各格子点に対してランダムなオフセット（ジッター）を加える手法を採用。
* **ランダム度の調整**: 0%（完全な格子状）から100%（最大級のばらつき）まで、用途に合わせて分布をコントロール可能。
* **スケールフリー設計**: 100mm角の小規模な面から1000mm以上の大きな面まで、サイズを問わず適切な密度で配置されます。
* **斜面・垂直面対応**: 面の法線を自動で計算し、あらゆる向きの平面に対して正確に投影・配置します。

## 🛠 使い方
1. 配置対象となる「面（Face）」を選択します（複数選択可）。
2. メニューの **[Plugins] > [精密ボロノイ母点生成 (V19)]** を実行します。
3. ダイアログで以下の値を設定します：
   - **分割密度**: 一辺あたりの分割数。値を大きくすると点が多くなります。
   - **ランダム度**: 0%で格子状、数値を上げるとばらつきが大きくなります。

## ⚠️ 注意事項
* **ファイルパスの制限**: 
  SVGインポーター同様、ファイル名やフォルダ名に日本語（全角文字）が含まれていると動作しない場合があります。半角英数字の環境での使用を推奨します。
